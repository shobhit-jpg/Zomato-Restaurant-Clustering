# Zomato Restaurant Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA), Sentiment Analysis, and Restaurant Segmentation using customer reviews and restaurant information from Zomato.

## Features

- Data Cleaning & Preprocessing
- Univariate, Bivariate & Multivariate Analysis
- VADER Sentiment Analysis
- Weighted Sentiment based on Reviewer Influence
- Restaurant Aggregation
- K-Means Clustering
- Restaurant Segmentation

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- NLTK
- VADER Sentiment Analyzer

## Dataset

### Restaurant Dataset
- Name
- Cost
- Collections
- Cuisines
- Timings

### Reviews Dataset
- Review
- Rating
- Reviewer
- Metadata
- Time
